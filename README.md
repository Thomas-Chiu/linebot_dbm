## [DEMO](https://www.youtube.com/watch?v=Z3ruSTqpgn0)

## v1.0.0

### Features

- 將小幫手加入 Line 群組或直接私訊點餐
- 輸入「訂便當」關鍵字啟動小幫手
- 輸入「結單」關鍵字統計每日便當數量、金額、備註及明細
- 輸入「棄單」關鍵字刪除今日訂單
